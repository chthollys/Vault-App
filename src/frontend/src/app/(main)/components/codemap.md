# Codemap

## Purpose
Shared storefront UI composition for home page and reusable game grid sections.

## Files
- `HomePage.tsx`: Client home container; fetches games, randomizes featured/hot/recommended sets, and renders loading fallbacks.
- `FeaturedGames.tsx`: Hero carousel for featured games with pricing overlay.
- `GameGridsSection.tsx`: Two `GameSection` blocks ("Hot Games" and "Recommended Games").

## Subdirectories
- `NavAsideBar`: Sidebar navigation/filter components and exports.
