# Codemap

## Purpose
Header controls for the all-games listing screen.

## Files
- `index.tsx`: Barrel exports for title module.
- `Title.tsx`: "All Games" section title component.
- `TitleSection.tsx`: Breadcrumb + sort select layout.
- `Selections.tsx`: Client sort select synced to `sortBy` URL query param.

## Notes
- Sort options include newest, popular, lowest-price, and highest-price.
